# 📝 To-Do List Web App

A clean, interactive, and fully responsive To-Do List web application built from scratch. This project helps users manage their daily tasks efficiently.

## 🔗 Live Demo
**[Click here to view the Live App](https://vikash-vi.github.io/todo-list-app/)**

## ✨ Features
- **Add Tasks:** Users can easily type and add new tasks to their list.
- **Delete Tasks:** A dedicated 'Delete' button for each task to remove it once completed.
- **Form Validation:** Alerts the user if they try to submit an empty task.
- **Clean UI:** A modern, card-based user interface with interactive hover effects.

## 🛠️ Technologies Used
- **HTML5:** For the basic structure and layout.
- **CSS3:** For styling, modern UI design, and responsive elements.
- **JavaScript (Vanilla JS):** For DOM manipulation, event listeners, and core logic.

## 👨‍💻 About This Project
This project was developed to practice and showcase fundamental web development skills, specifically how HTML, CSS, and JavaScript interact with each other to create a functional web application.
